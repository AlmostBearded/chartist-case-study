Chartist Case Study - Sizing

These examples experiment with different chart sizing variants.

Default Size

Neither width nor height are specified. The sizing of this depends on the sizing of the ct-chart class (if specified).

Fixed Size

Both width and height are specified in absolute units.

Fixed Height

Only height is specified in absolute units. The width of the chart is fluid.

Aspect Ratio

Both width and height are fluid but a specific aspect ratio between them is maintained.
